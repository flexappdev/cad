## What it is

CAD ships **one fresh course per day** on any topic, generated by Claude
and persisted forever. A course is **10–15 lessons + a 5-question capstone
quiz**, with an AI-generated cover (FLUX) and intro video loop (Seedance).
The library at `/library` is browsable, searchable, and grows by exactly
one course per day.

**Positioning** — hybrid. The engine is **generic** (any topic, autonomous
topic-picker that dedupes against history). Days 1–30 are **seeded** with
eight business-English courses ported from `~/courses/_original/` so the
library opens with real content from day zero.

## Quick start

```bash
cd ~/APPS/cad
pnpm install
cp .env.local.example .env.local      # fill in MONGODB_URI, ANTHROPIC_API_KEY, etc.

pnpm seed                              # ingest 8 business-English seed courses
pnpm daily:dry                         # generate today's course, no writes
pnpm daily                             # generate + write Mongo + S3
pnpm dev                               # http://localhost:15011
pnpm test                              # vitest

# fleet skills
/abc-vercel link cad
/abc-supabase sync env cad
/abc-mongo ping cad
/abc-ga sync cad G-XXXXXXXXXX
/abc-fleet-loop register cad daily 00:05
```

## Heritage

CAD is inspired by — but does not clone — `~/courses/`, a 2025-era
FastAPI + Vite English-teacher system. It donates:

- **Course content shape** (Course + Sessions + key-ideas).
- **Seed material** (8 business-English outlines under `_original/`).
- **Markdown → JSON parser** logic (Python → ported to TS).

The Python backend is **not** carried forward — CAD unifies on the fleet's
proven Next.js stack.
