## PROPOSAL ‚Äî content / marketing / monetisation

### Content

- **Daily generic course** ‚Äî Claude picks a never-done-before topic at 00:05 UTC; outputs 10‚Äì15 lessons + 5-question capstone.
- **Day 1‚Äì30 seed** ‚Äî 8 business-English courses from `~/courses/_original/` ingested as `seed: true` rows, plus the first 22 days of organic daily generation, so the library has 30+ courses at end of month 1.
- **Topic discipline** ‚Äî Claude is given a dedupe list (last 90 days). For the first 30 days a "topic hint" can be passed from `/admin` (e.g. "lean toward soft skills this week") to balance the library across business / tech / language / soft-skill / general-knowledge clusters.
- **No instructors** ‚Äî courses are bylined "CAD AI" v1; consider a "founder picks" badge for the 8 ported seed courses.

### Marketing

- **SEO long-tail** ‚Äî title + tagline + lesson body are stable URLs; sitemap.xml grows by one entry/day.
- **RSS** ‚Äî `/feed.xml` for daily syndication. Course-A-Day is positioned as a free daily learning newsletter (without the newsletter at v1).
- **Daily digest (v2)** ‚Äî Resend-delivered email summary of today's course; opt-in via `/login` page CTA.
- **Cross-link from LAD / IAD / PAD** ‚Äî fleet siblings link to CAD in their footers; CAD links back. Easiest organic boost.

### Monetisation

- **v1** ‚Äî free, no paywall, no ads.
- **v2** ‚Äî Resend daily digest (free signup, captures email for product analytics).
- **v3** ‚Äî Stripe Pro tier (¬£3/mo): TTS audio per lesson, printable PDF, quiz certificate. Single tier, no annual at first.
- **Affiliate layer** ‚Äî book recommendations per topic (Amazon Associates `fs08-21` ‚Äî the existing fleet tag from FS).

**Day-1 cost** ‚Äî ~$0.18/day (Claude + Runware). Well under the fleet $1/day/site monetisation cap.
